# Retail Sales Analysis and Sales Prediction using Machine Learning

## Project Overview

This project focuses on analyzing retail sales data and predicting future sales using Machine Learning. The objective is to understand sales patterns, identify profitable business areas, and build a predictive model that can estimate sales based on historical data.

The project includes data preprocessing, exploratory data analysis (EDA), visualization, model training, and performance evaluation using Python.

---

## Dataset

**Dataset Name:** Sample Superstore Sales Dataset

**File Format:** Excel (.xls)

The dataset contains information about:

* Order ID
* Order Date
* Product Category
* Product Sub-Category
* Customer Segment
* Region
* Sales
* Profit
* Discount
* Order Quantity
* Shipping Cost
* Unit Price

---

## Project Workflow

1. Load the dataset.
2. Clean and preprocess the data.
3. Handle missing values.
4. Convert date columns into Year and Month.
5. Encode categorical variables.
6. Perform Exploratory Data Analysis (EDA).
7. Create visualizations.
8. Train a Linear Regression model.
9. Evaluate the model using MAE, MSE, RMSE, and R² Score.
10. Generate business insights.
